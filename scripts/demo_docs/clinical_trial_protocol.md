# 臨床試験プロトコル — 第II相試験

## 研究概要

本プロトコルは、新規抗線維化化合物SYN-447の安全性および有効性を評価するための多施設共同第II相無作為化比較試験を定義するものである。対象疾患は特発性肺線維症（IPF）とし、主要評価項目は24週時点における努力肺活量（FVC）の変化率とする。

## 適格基準

組み入れ基準:
- 年齢18歳以上80歳以下
- 高分解能CT所見および外科的肺生検によりIPFと確定診断されていること
- スクリーニング時のFVCが予測値の45%以上であること
- 書面による同意能力を有すること

除外基準:
- 過去12ヶ月以内の急性増悪の既往
- 重篤な腎機能障害（eGFR < 30 mL/min/1.73m²）
- 妊娠中または授乳中の女性
- 強力なCYP3A4阻害薬の併用

## 投与レジメン

SYN-447は経口投与とし、1日2回150mgを食後に服用する。用量調整は治験責任医師の判断により50mg単位で行うことができる。プラセボ群は外観を一致させた錠剤を同一スケジュールで投与する。

治験期間は24週間の投与期間と8週間の追跡期間からなる。有害事象はCTCAE v5.0に従ってグレード分類し、Grade 3以上の事象は48時間以内に治験依頼者に報告する。

## 統計解析計画

主要解析はintent-to-treat集団を対象とし、混合効果反復測定モデル（MMRM）を用いる。共変量にはベースラインFVC、年齢、性別、地域（日本/欧州）を含める。優越性の検定水準はα=0.05（両側）とし、検出力80%を確保するため各群42例、計84例を目標症例数とする。欠測データは多重代入法により処理する。

## 安全性モニタリング

独立データモニタリング委員会（IDMC）が12週ごとに中間解析を実施する。事前規定の中止基準は、いずれかの群においてGrade 4以上の薬物関連有害事象が3例以上発生した場合、または死亡が2例以上発生した場合とする。
